# 微创AI带货视频工坊 — UI/UX 设计方案

---

## 一、设计哲学

### 1.1 核心理念：**「丝滑流水线」**

```
素材 ▸ 脚本 ▸ 分镜 ▸ 生图 ▸ 成片
  └──── 一气呵成，每步可回溯 ────┘
```

用户需要的不是一个功能堆砌的工具，而是一条**无摩擦的创作流水线**。每个环节的输出自然流入下一环节，同时允许随时回退修改。

### 1.2 五大设计原则

| 原则 | 含义 | 执行标准 |
|------|------|---------|
| **零学习曲线** | 新手 3 分钟内完成第一条视频 | 关键路径 ≤ 3 次点击，引导不超过 2 步 |
| **所见即所得** | 所有编辑实时预览 | 延迟 < 200ms 的操作必须即时反馈 |
| **渐进式复杂度** | 默认隐藏高级选项，按需展开 | 一屏核心控件 ≤ 7 个 |
| **AI 透明化** | AI 做什么、做了多久、可以怎么改，始终可视化 | 所有 AI 操作有进度、有预览、可撤销 |
| **暗色沉浸** | 创意工具的默认暗色模式 | #0A0A0F 主背景，低刺激，内容发光 |

### 1.3 情绪板关键词

`暗色` `玻璃质感` `微光` `锐利` `流动` `精准` `轻量` `呼吸感`

### 2.2 字体系统

| 用途 | 字体 | 字重 | 字号 | 行高 |
|------|------|------|------|------|
| 页面标题 | Inter | 700 | 28px | 1.3 |
| 模块标题 | Inter | 600 | 20px | 1.4 |
| 卡片标题 | Inter | 600 | 16px | 1.4 |
| 正文 | Inter | 400 | 14px | 1.6 |
| 辅助说明 | Inter | 400 | 12px | 1.5 |
| 代码/数据 | JetBrains Mono | 400 | 13px | 1.5 |

### 2.3 间距系统（4px 基栅）

```
│  xs  │  sm  │  md  │  lg  │  xl  │ 2xl  │ 3xl  │
│  4   │  8   │  12  │  16  │  24  │  32  │  48   │
```

### 2.4 圆角系统

| Token | 值 | 用途 |
|-------|-----|------|
| radius-sm | 6px | 标签、小按钮、输入框 |
| radius-md | 10px | 卡片、面板、模态框 |
| radius-lg | 16px | 大容器、视频预览框 |
| radius-full | 999px | 药丸按钮、头像、状态指示器 |

## 八、微交互定义

| 交互 | 触发 | 反馈 | 时长 |
|------|------|------|------|
| 分镜卡片拖拽 | Drag start | 卡片升起 8px + 阴影扩大 + 紫色描边 | 150ms |
| 分镜卡片放下 | Drop | 卡片弹回平面 + 其他卡片滑开让位 | 250ms spring |
| 版本切换 | Click | 旧版淡出 ← 新版滑入（方向感知） | 200ms |
| AI 生成完成 | 任务结束 | 结果从底部滑入 + 轻微弹跳 + 成功音效 | 400ms |
| 步骤切换 | Click step | 内容区横向滑动切换，方向=步骤方向 | 300ms ease-out |
| 删除确认 | Click delete | 卡片缩小消失 + 后续卡片上移填补 | 200ms |
| 导出按钮 | Hover | 渐变色从左到右流动一次 | 800ms |
| 通知提示 | 事件触发 | 右上角滑入 → 停留 3s → 淡出 | 300ms in, 3s stay |

## 九、无障碍设计

| 标准 | 实现 |
|------|------|
| 色彩对比度 | 所有文本 ≥ WCAG AA (4.5:1)，大文本 ≥ 3:1 |
| 键盘导航 | 完整 Tab 序，Enter/Space 激活，Esc 关闭，方向键在时间线中移动 |
| 屏幕阅读器 | 所有图标有 aria-label，AI 状态有 live region 播报 |
| 焦点指示 | 2px 紫色外框 + 2px 偏移，所有可交互元素可见焦点 |
| 减速动画 | `prefers-reduced-motion` 时禁用所有非必要动画 |
