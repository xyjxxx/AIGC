# 微型一站式AI创作带货视频平台 — 产品需求文档 (PRD)

---

## 一、产品概述

### 1.1 产品名称
**微创AI带货视频工坊**（暂定）

### 1.2 产品定位
面向中小电商卖家、个人创业者、短视频带货达人的**零门槛AI视频创作工具**。用户只需导入商品素材（图片、文字、链接），平台自动完成脚本撰写→分镜设计→分镜图生成→成品视频输出的全链路创作。

### 1.3 目标用户

| 用户角色 | 特征 | 核心痛点 |
|---------|------|---------|
| 电商小卖家 | 淘宝/拼多多/抖音小店店主，人少预算低 | 不会写脚本、不会拍摄、请不起团队 |
| 带货达人 | 短视频带货新手，想批量产出内容 | 单条视频制作耗时、创意枯竭 |
| 品牌方市场人员 | 有产品但缺乏视频营销能力 | 外包视频贵且沟通成本高 |
| MCN机构 | 批量孵化账号，需要标准化产出 | 视频产能瓶颈、质量不稳定 |

### 1.4 核心价值主张
> **"上传商品素材，一键生成专业带货视频。"**
- **省时**：从几天缩短到几分钟
- **省力**：无需文案、拍摄、剪辑技能
- **省钱**：单条视频成本趋近于零
- **专业**：AI理解商品卖点，生成营销级别的内容

### 模块一：视频文案读取创作脚本

#### 2.1.1 功能概述
用户导入商品信息后，AI自动分析并生成多个版本的带货脚本，用户可以选择、编辑、拼接，最终确定拍摄脚本。

#### 2.1.2 输入方式（5种素材入口）

| 输入方式 | 说明 | 适用场景 |
|---------|------|---------|
| **商品链接** | 粘贴电商平台商品链接，自动抓取标题/详情/参数/评价 | 有现成商品链接的用户 |
| **上传图片** | 上传商品图片，AI视觉识别商品特征 | 无法提供链接的用户 |
| **手动输入** | 输入商品名称+核心卖点（≥20字） | 快速尝试 |
| **PDF/文档上传** | 上传商品介绍文档、品牌手册 | 品牌方、有完整素材的 |
| **批量导入** | CSV批量导入多款商品 | 店铺批量制作 |

#### 2.1.3 AI脚本生成引擎

**脚本模板体系**（6大类，可扩展）：

| 模板类型 | 适用品类 | 节奏特点 | 时长 |
|---------|---------|---------|------|
| **痛点解决型** | 家居/个护/工具 | "你有没有这个痛点→我们解决了" | 30-60s |
| **测评对比型** | 数码/美妆/食品 | "A vs B→实测对比→结果反转" | 45-90s |
| **使用教程型** | 小家电/配饰/DIY | "开箱→步骤演示→效果展示" | 60-120s |
| **场景种草型** | 服饰/香薰/零食 | "氛围感场景→产品融入→心动" | 15-45s |
| **口播种草型** | 全品类 | "达人视角→真实体验→推荐理由" | 30-60s |
| **促销快闪型** | 大促/清仓/爆品 | "快节奏→价格优势→紧迫感" | 15-30s |

**脚本输出结构**：
```
├── 标题（3-5个备选）
├── 开场（黄金3秒抓注意力）
├── 正文（卖点逐个展开）
│   ├── 卖点1 + 画面描述
│   ├── 卖点2 + 画面描述
│   └── 卖点3 + 画面描述
├── 结尾（促单/引导下单）
└── 标签建议（#话题标签）
```

**AI模型方案 — 用户自选平台（全模块统一）**：
- 平台提供 **OpenAI** 和 **字节豆包(Doubao)** 两种AI平台供用户选择，选定后全模块统一使用
- **脚本生成**、**分镜图生成**、**视频AI优化** 三大模块全部使用用户所选平台
- 用户登录自有账号授权（OpenAI账号 或 字节跳动账号），费用由用户API账户承担
- 平台仅做请求代理和结果解析，不存储用户API密钥（加密传输）
- 后续可在设置中随时切换平台

**AI能力要求**：
- 商品卖点提取（基于图片+文本多模态）
- 目标受众适配（可指定人群：宝妈/学生/上班族等）
- 平台风格适配（抖音/快手/小红书/视频号 差异化脚本）
- 合规检测（避免违反广告法词汇如"最""第一"等）

#### 2.1.4 脚本编辑器功能

| 功能 | 说明 |
|------|------|
| **多版本对比** | 一次生成3个版本，并排对比查看 |
| **段落编辑** | 支持拖拽排序、新增、删除、合并段落 |
| **AI改写** | 选中段落→"更有趣"/"更专业"/"更简短"/"更口语化" |
| **字数/时长预估** | 实时显示总字数和预估视频时长 |
| **版本历史** | 自动保存编辑历史，支持回退 |
| **团队协作** | 分享链接给团队成员审阅评论 |

#### 2.1.5 输出
- 完整脚本（含画面描述、口播文案、时长标注）
- 自动流转至下一模块（分镜画板）

### 模块二：视频文案读取创作画板（分镜设计）

#### 2.2.1 功能概述
将已生成的脚本自动拆解为分镜序列，提供可视化画板供用户调整分镜节奏和构图。

#### 2.2.2 自动分镜拆解

**拆解规则**：
- 每个卖点段落 → 1-N个分镜（根据段落长度自适应）
- 开场/结尾单独成镜
- 产品特写、功能演示、使用场景 自动标注

**分镜卡信息字段**：
```
┌─────────────────────────┐
│ 分镜编号：S01           │
│ 时长：3秒               │
│ 景别：特写/近景/中景/远景│
│ 运镜：固定/推/拉/摇/跟  │
│ 画面描述：双手展示产品.. │
│ 对应口播：这样的话...    │
│ 转场效果：溶解/切/滑动   │
└─────────────────────────┘
```

#### 2.2.3 画板编辑器

| 功能 | 说明 |
|------|------|
| **时间线视图** | 横向时间轴，每个分镜为可拖拽卡片 |
| **分镜增删改** | 添加新分镜、删除、修改分镜参数 |
| **拖拽排序** | 调整分镜顺序 |
| **时长调整** | 拖拽分镜边缘调整时长，总时长实时更新 |
| **批量设置** | 统一设置景别风格/转场类型 |
| **预览模式** | 静态画面序列预览（占位图） |
| **分镜模板** | 保存常用分镜组合为模板，一键复用 |

#### 2.2.4 智能优化建议
- 节奏检测：分镜时长分布是否合理（避免单调）
- 景别建议：提醒近/中/远景的合理搭配比例
- 转场推荐：根据前后分镜内容推荐最合适的转场

#### 2.2.5 输出
- 完整分镜列表（含所有元数据）
- 自动流转至下一模块（分镜图生成）

### 模块三：根据视频文案脚本生成可使用的脚本分镜图

#### 2.3.1 功能概述
基于每个分镜的"画面描述"，AI生成对应的分镜图/关键帧。支持风格选择、角色一致性保持和逐图微调。

#### 2.3.2 AI生图引擎
> **与脚本/视频共用同一AI平台**：用户选择的 Codex/豆包平台统一驱动生图，无需额外配置。

**支持的图像风格**：

| 风格 | 适用场景 | 代表效果 |
|------|---------|---------|
| **真实摄影** | 全品类通用 | 产品实物感、真人感 |
| **3D渲染** | 数码/家居/汽车 | 高级质感、光影细腻 |
| **扁平插画** | 知识付费/教育 | 亲和力、简洁 |
| **国潮风** | 国货/食品/文创 | 中国元素、潮流感 |
| **极简白底** | 标准产品展示 | 干净、电商感 |
| **动漫风** | 二次元/潮流 | 年轻化 |

**角色一致性方案**：
- 用户上传品牌IP/虚拟人形象 → 全程绑定
- 未上传 → AI自动生成统一的人物形象（保持面部、服装一致）
- 支持"参考图模式"：上传一张图作为风格/人物参考

#### 2.3.3 分镜图提示词生成

**自动提示词构建逻辑**：
```
输入：分镜画面描述 + 风格选择 + 角色参考
    ↓
提示词模板引擎：
  - 主体描述（商品/人物） 
  - 场景描述（背景/环境）
  - 镜头描述（景别/角度）
  - 风格描述（所选风格+画质关键词）
  - 灯光/色调描述
  - 负向提示词（变形/模糊/多手指等）
    ↓
输出：完整AI绘画提示词
```

#### 2.3.3 分镜图编辑

| 功能 | 说明 |
|------|------|
| **图片画廊** | 所有分镜图呈现在画廊中，点击放大 |
| **单图重生成** | 不满意的单张图片→重新生成（3个变体） |
| **局部重绘** | 圈选图片中不满意区域→描述修改→局部重绘 |
| **批量生成** | 一键生成所有分镜图（后台排队） |
| **提示词编辑** | 高级用户可直接编辑prompt |
| **图库替换** | 从内置素材库/用户上传的图中替换 |

#### 2.3.5 素材库
- 内置免版权场景图库（办公/家居/户外/城市...）
- 内置免版权人物图库（各年龄段/职业/风格）
- 用户可上传自有素材
- 历史生成图片自动存档

#### 2.3.6 输出
- 每个分镜对应的成品图（高清）
- 自动关联至下一模块

### 模块四：生成视频

#### 2.4.1 功能概述
将分镜图和脚本合成为完整视频，包含配音、背景音乐、字幕、转场、特效等。
> **AI调用**：视频AI优化任务（字幕纠错、转场推荐、卖点识别、节奏分析）统一使用用户已选的 Codex/豆包平台，与脚本、生图共用同一账号授权。

#### 2.4.2 视频合成引擎

**合成流程**：
```
分镜图序列 + 脚本/口播文本
    ↓
① TTS语音合成 → 音频轨道
② 图片适配（Kenhance/缩放/构图优化）
③ 时间轴编排（分镜图时长对齐音频）
④ 转场特效插入
⑤ 字幕生成（语音转文字对齐）
⑥ 背景音乐混音
⑦ 品牌水印/片头片尾
    ↓
成品视频输出
```

#### 2.4.3 TTS配音

| 功能 | 参数 |
|------|------|
| **音色选择** | 男女各10+种音色（含方言：东北/四川/粤语） |
| **语速调节** | 0.5x - 2.0x |
| **语调调节** | 平铺直叙/活泼/沉稳/激昂 |
| **分段配音** | 不同分镜可配不同音色（对话感） |
| **多语言** | 中文/英文/中英混读 |
| **自录音频** | 支持上传自己的录音替代TTS |

#### 2.4.4 字幕系统

| 功能 | 说明 |
|------|------|
| **自动生成** | 语音自动转字幕，时间轴自动对齐 |
| **字幕样式** | 字体/颜色/大小/描边/阴影/背景条/位置 |
| **字幕模板** | 10+套带货风格字幕模板一键套用 |
| **关键词高亮** | 自动识别促销词（限时/特价等）添加高亮动画 |
| **手动编辑** | 修改字幕文字、调整时间轴 |

#### 2.4.5 背景音乐

| 来源 | 说明 |
|------|------|
| **内置曲库** | 按情绪/风格/时长分类的免版权BGM |
| **AI生成** | 输入风格→AI生成专属BGM |
| **本地上传** | 用户上传自己的音乐 |
| **智能音量** | 配音时自动降低BGM音量（闪避效果） |

#### 2.4.6 转场与特效

- 10+基础转场（淡入淡出/滑动/缩放/旋转/模糊/翻页等）
- 智能转场推荐（根据相邻分镜内容推荐）
- 关键帧动画（图片缩放/位移动画 即 Ken Burns 效果）

#### 2.4.7 视频输出规格

| 参数 | 选项 |
|------|------|
| **分辨率** | 720P / 1080P / 2K / 4K |
| **比例** | 16:9横屏 / 9:16竖屏 / 1:1方形 |
| **帧率** | 24fps / 30fps / 60fps |
| **格式** | MP4 / MOV |
| **码率** | 自动 / 手动设定 |
| **平台预设** | 抖音(9:16 1080P) / 小红书(3:4) / B站(16:9) |

#### 2.4.8 预览与快速修改
- 低分辨率实时预览（带时间轴）
- 逐帧微调（调整单帧显示时长）
- 快速替换某分镜图/某句配音
- A/B版本对比

### 3.2 核心技术选型

| 层级 | 技术 | 说明 |

| 容器化 | Docker + K8s | 部署运维 |

## 四、用户流程

### 4.1 核心创作流程

```
1. 创建项目
   ├── 命名项目
   ├── 选择目标平台（抖音/快手/小红书...）
   └── 选择视频时长目标

2. 配置AI平台（首次使用）
   ├── 选择AI平台：OpenAI(Codex) / 字节豆包(Doubao)
   │   └── 应用于全部模块：脚本生成 + 分镜生图 + 视频AI优化
   ├── 登录对应平台账号并授权
   ├── 平台加密存储授权令牌（不触碰用户密码）
   └── 后续可随时切换平台

3. 输入素材
   ├── 粘贴链接 / 上传图片 / 手动输入 / 批量导入
   └── AI自动解析商品信息

4. 生成脚本（模块一）
   ├── 选择模板风格
   ├── AI生成3个版本
   ├── 用户选择/编辑/拼接
   └── 确认脚本 → 下一步

5. 设计分镜（模块二）
   ├── AI自动拆解分镜
   ├── 用户调整分镜序列和时长
   ├── 设置景别/运镜/转场
   └── 确认分镜 → 下一步

6. 生成分镜图（模块三）
   ├── 选择图像风格
   ├── 上传角色参考（可选）
   ├── AI批量生成分镜图
   ├── 逐图审核和微调
   └── 确认分镜图 → 下一步

7. 合成视频（模块四）
   ├── 选择音色 → TTS生成配音
   ├── 选择BGM
   ├── 选择字幕样式
   ├── 预览视频
   ├── 微调
   └── 导出/发布
```

### 4.2 辅助流程

```
- 项目库管理：查看/复制/删除历史项目
```

## 八、风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|---------|
| AI生成质量不稳定 | 用户体验差 | 多模型冗余+用户手动微调+持续训练优化 |
| 生成速度慢 | 用户流失 | 异步任务+排队提示+预生成+模型加速 |
| 版权纠纷 | 法律风险 | 免版权素材库+AI生成原创性确保+免责条款 |
| 同质化竞争 | 市场份额 | 垂直深耕带货场景+模板生态壁垒+渠道绑定 |
| 算力成本高 | 利润压力 | 模型蒸馏+边缘计算+动态调度 |
| 内容合规 | 平台处罚 | 广告法检测+敏感词过滤+人工审核 |
